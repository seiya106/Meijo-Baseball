# Meijo Baseball Dataset
Due to the large file size of the data used in this project, it is shared via Google Drive instead of GitHub.

## 📊 Dataset Details
This dataset includes two main components: `baseball_s` and `baseball_h`.

`baseball_s`: Includes 3 pitch types: Curve (CV), Fork (FK), and Straight (ST).

`baseball_h`: Includes 3 pitch types: Cut (CT), Curve (CV), and Straight (ST).

### Structure

```text
dataset/
├── baseball_s/ 
│   └── default/
│       ├── CV/ <- 40 samples, 64 frames + videos (40 Raw videos)
│       ├── FK/ <- 40 samples, 64 frames + videos (40 Raw videos)
│       ├── ST/ <- 40 samples, 64 frames + videos (40 Raw videos)
│       └── README.txt   
└── baseball_h/ 
    └── default/
        ├── CT/ <- 41 samples, 64 frames + videos (41 Raw videos)
        ├── CV/ <- 41 samples, 64 frames + videos (41 Raw videos)
        ├── ST/ <- 41 samples, 64 frames + videos (41 Raw videos)
        └── README.txt
```
